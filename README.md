# Microscopic Railway Network Simulation

## Project Overview
This repository contains a microscopic rail transport simulation developed using OpenTrack. The project focuses on modeling railway infrastructure, signaling systems, and train timetables to evaluate network capacity and operational efficiency.

## Key Engineering Tasks
* **Infrastructure Modeling:** Designed a digital twin of a rail network, incorporating precise track geometries, stations, and junction routing.
* **Signaling & Safety Systems:** Configured block sections, signal locations, and speed restrictions to simulate safe train separation and headway calculations.
* **Operations & Capacity Analysis:** Simulated scheduled train movements to evaluate line capacity, identify network bottlenecks, and optimize passenger transit times.
